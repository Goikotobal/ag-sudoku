# AG Sudoku v1.0.1

AI-powered Sudoku puzzle game with 12 language support.

**Live:** https://sudoku.alexgoiko.com

## Features
- 3 difficulty levels (Medium, Expert, Pro)
- AI-powered hints with educational explanations
- Smart notes system
- Auto-save functionality
- Offline play
- 12 languages: English, Spanish, German, French, Italian, Portuguese, Japanese, Korean, Chinese, Hindi, Tagalog, Basque

## Tech Stack
- Next.js 14
- TypeScript
- next-intl (internationalization)
- React

## Development
```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Production
```bash
npm run build
npm start
```
